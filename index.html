⚙️ 初始設定精靈
ℹ️ 第一次使用需要完成以下設定，之後就不需要再設定了。
1

上傳三個 DOC 模板到 Google Drive

把以下三個原始 DOC 檔案上傳到你的 Google Drive 任一資料夾：
奪標-國小組.doc　恆春-國小組.doc　洋蔥-國小組.doc

2

取得每個 DOC 的 File ID

在 Drive 右鍵每個 DOC → 取得連結 → 複製
連結格式：https://drive.google.com/file/d/這段是FileID/view

3

在 GAS 執行 setTemplates()

在 GAS 編輯器上方函式選單選 setTemplates，或在底部輸入下面的指令後點執行：

setTemplates(
  "奪標 DOC 的 FileID",
  "恆春 DOC 的 FileID",
  "洋蔥 DOC 的 FileID"
)
4

驗證設定是否成功

回到「單筆填寫」頁面，填入一筆資料試產生，若成功即代表設定完成。
