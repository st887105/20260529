⚙️ 初始設定精靈
ℹ️ 第一次使用需完成以下設定，之後不需要重複設定。
1

執行 setupDatabase（只需一次）

在 GAS 編輯器上方函式選單選 setupDatabase，點執行，初始化記錄工作表。

2

上傳三個 DOC 模板到 Google Drive

把下列三個原始 DOC 上傳到你的 Google Drive：
奪標-國小組.doc  恆春-國小組.doc  洋蔥-國小組.doc

3

取得每個 DOC 的 File ID

在 Drive 右鍵每個 DOC → 取得連結 → 複製
連結格式：https://drive.google.com/file/d/這段是FileID/view

4

填入 File ID，完成設定

setTemplates(
  "奪標 FileID",
  "恆春 FileID",
  "洋蔥 FileID"
)
5

驗證設定完成

前往「單筆填寫」試產生一筆，成功代表設定完成！
